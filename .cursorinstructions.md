# 🤖 Инструкции для Cursor AI

## О проекте
Это онлайн-школа "Династия" для подготовки к ЕГЭ/ОГЭ.

## Важные правила
1. **НЕ коммить** файлы `.env` в Git
2. **НЕ менять** структуру базы данных без миграций
3. **ВСЕГДА** использовать TypeScript
4. **ВСЕГДА** проверять адаптивность на мобильных

## Паттерны кода
- API endpoints в `server/src/routes/`
- Компоненты в `client/src/components/`
- Страницы в `client/src/pages/`
- Prisma схема в `server/prisma/schema.prisma`

## Тестирование
- Frontend: `cd client && npm run dev`
- Backend: `cd server && npm run dev`
- База: Supabase Dashboard

## Частые команды
```bash
# Запустить всё
cd client && npm run dev &
cd server && npm run dev

# Git
git add .

git push

# Prisma
